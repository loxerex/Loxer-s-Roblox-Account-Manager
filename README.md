# Loxer's Roblox Account Manager

**LXAM** is a lightweight Roblox account manager currently in alpha.  
Built to make managing multiple Roblox accounts simple, fast, and clean.

## Features

- Easy account importing
- Built-in multi instance support
- Personal game catalogue
- Lightweight and optimized

## Performance

LXAM is designed to be extremely lightweight.

- ~40MB memory usage
- Near-zero CPU usage while idle
- `_internal` folder is required for resources and runtime files

## Usage

1. Download and unzip the release
2. Run the main `.exe`
3. Do not remove the `_internal` folder

## Current Status

LXAM is currently in **alpha**, so bugs and missing features are expected.

More features and improvements are planned.
